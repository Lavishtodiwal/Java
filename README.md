# Java Practice Repository

This repository is a collection of Java programs and examples demonstrating core Java concepts, object-oriented principles, and hands-on practice with real code. It’s structured into packages/folders to help you explore Java topics one at a time.

## 📌 Overview

This project includes sample code on:

- Basic language constructs and arrays  
- Object-oriented programming (OOP) concepts  
- Exception handling  
- Input/Output  
- Collections framework  
- Graphical user interface (AWT)  
- JDBC database connectivity  
- Servlets  
- XML handling  
- And various small practice programs

Each folder contains relevant `.java` files showcasing patterns and concepts.
